TABLEAU

Above is an interactive dashboard that shows how customers rate flights - satisfied or dissatisfied/neutral. There are 4 tabs in the dashboard, with the heatmap splitting satisfaction by reason for travel and class of travel on every tab as a benchmark and filter. By clicking anywhere on this heatmap, the other graphs will adjust to show data from only the section selected in the heatmap. The tabs have graphs that correspond to factors that work together, for example, all graphs regarding inflight service are grouped to one tab.

Above is a dynamic dashboard showing flights leaving the three major airports around New York City - JFK, LaGuardia, and Newark. There are three charts on the dashboard, with the first being a list of destinations with a bar showing the count of flights to that location. The second chart is a stacked bar chart per airline, where number of flights departing from each airline are shown in colors. Destinations are marked on the map, with size and color of the dot reflecting the popularity of the destination. By clicking on a circle, the data on the other two charts will focus to the destination selected.

Above is a dynamic dashboard showing the names that have applied to the social security office. As this is usually done at birth, it can be used to see the popularity of baby names. The default screen is a list of all names ever used along with the count of how many times they were used on the left, along with a graph of the count of name applications per year. These charts can be filtered by the three filters on the right - gender, range of years, and a single select for name. The single select is so that the popularity of a singular name over time can be seen. For that filter, it is probably more efficient to use the search functionality on the name filter which can be foung by clicking on the magnifying glass that appears when hovering over the title of the name filter.

ChatGPT was used to create an interactive sorting hat - the first question relates to age because the questions that follow are age relative. The questions are situation based, with the user having to type in their response and why, with the AI then breaking down your response into the house traits.
